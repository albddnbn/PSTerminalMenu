PS1 Menu...with Functions! Change Log

September 9, 2023

  • Removed:The Browser Functions category, renamed the text file to .old. Functions are never used.

    • The actual function files are still there, but the category won't show in menu until it's readded to categories.txt, and the Browser Functions.txt.old file has .old removed.


  • Removed:Install-Teams, Install-Zoom, Install-LoggerPro from the installation.txt file. They won't show in menu until the funciton names are re-added to installation.txt.

    • These functions are never used, and the Ps App toolkit deployments basically take care of all of them.


  • Modified:The try-catch section of menu.ps1. Before, it would stop dot-sourcing functions as soon as it ran into an error.

    • Now it will write a notification to the terminal in red, wait 3 seconds, then continue dot sourcing the rest of the functions if there are any.

June 25, 2023

  • Added: A way for users to request the description of a parameter being prompted for parameter value.

    • Used Get-Help to access the parameter descriptions, so the function has to have those in order for a parameter's description to be printed to terminal.

  • Improved performance
