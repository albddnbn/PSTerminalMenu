Powershell menu...with functions!

add category add function

Prints an interactive menu to the terminal where IT person can:

  1. Select a category of function (this is done so there aren't 20+ functions printed to terminal menu at once)
  2. Select function to run
  3. Script will prompt for values for any existing parameters
    • Right now, the script doesn't prompt for the common variables (see comments in script, it'd be good to improve that whole section to deal with more complex functions)

How to add a category to the menu

Example below: Adding the category - Browser Functions - to the menu.

  1. Type the category name into 'categories.txt'. Since I want the Browser option to show directly below Information Gathering, that's where I put it in the text file.

  2. Create a new text file in the functions directory with the same name as the category. In this case, I created a text file called 'browser functions.txt'.

  3. At this point, try bringing up the menu to make sure your new category is showing.

    4. How to add a function to the menu

    Example below: Adding a function (Install-GenericMSI in this case) to the menu.

  1. Now I'm going to put a new Get-ChromeExtensions.ps1 file/function into that category. Since the function is a pretty long one, I'm going to create a file called Get-ChromeExtensions.ps1 in the same directory as menu.ps1 and put the Get-ChromeExtensions function inside that.
  2. Lastly, write the name of the file/function into it's corresponding category text file. In this case, I'm adding 'Get-ChromeExtensions' to 'Browser Functions'.
