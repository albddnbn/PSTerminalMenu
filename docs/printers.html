Generate exe to connect user to printer:

Build-PrinterConnectionExe allows you to create an executable that will connect a user to either...

  1. a PrinterLogic printer (stored in your printercloud)
  2. a printer on a traditional print server

Spreadsheet showing users and their printers:

Get-ConnectedPrinters will check target computer(s) for users logged in, and attempt to get a list of their connected and default printers.

Installing PrinterLogic client software:

Use Install-Application > PrinterLogic to install 'Printer Installer Client' software on target computers using the 'PrinterLogic' folder in 'deploy/applications'.
