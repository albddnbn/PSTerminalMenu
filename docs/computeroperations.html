Corrupt profiles and temporary user folders:

Clearing Firefox profiles:

Clear-FirefoxProfile can be used to resolve a common Firefox issue that arises when a user's firefox profile is missing or inaccessible.

More info: https://support.mozilla.org/en-US/kb/how-run-firefox-when-profile-missing-inaccessible

Clear-FirefoxProfile will delete the user's profiles.ini file, with option to delete the user's entire firefox profile folder, which should allow them to restart Firefox and load a/their profile successfully.

Sending files to and from computers:

Send-Files: is a function that allows you to pick a file/folder on local computer, and send it to target filepath on remote computer(s).

Copy-Remotefiles: allows you to specify a local filepath to look for on target remote computers. If found, script will copy file/folder back to local computer in specified output folder.

Format secondary storage on target computer(s):

At times, a computer will have it's main storage drive where the OS is installed, and a secondary storage drive.

For different reasons - students and instructors may have need of storage beyond the capacity of the main storage drive.

Format-SecondaryDrive allows you to initialize and format any secondary drives on target systems.

The script will take a disk inventory of one of the computers in the target list and present menu to the user where they can choose the disk they'd like to format and other options.

For the reason above, all computers in the target list have to have the same kind of secondary storage.

Ex: I had a classroom full of new Optiplex 7660's that all had the same secondary storage / SSD drive, I used this function to format the drives quickly to make them accessible to students/instructors.

Create executable to map shared folder for a user:

New-DriveMappingExe: lets you specify the UNC path of a shared folder, and the drive letter you'd like for it to be assigned when mapped.

Then it uses the PS2exe module to compile an executable from a powershell script that tries to map the folder for whoever double-clicks it.

Copy-Remotefiles: allows you to specify a local filepath to look for on target remote computers. If found, script will copy file/folder back to local computer in specified output folder.

Turn off and optionally disable Wi-Fi adapters:

Sometimes computers with existing/active wired connections to the network can also have active wireless connections, which can impact a computer's performance for end-user.

Stop-WifiAdapters will disable and stop any wireless adapters on target computers, but only if the computer has active Ethernet connection.
