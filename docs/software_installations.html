Software Installations

Author : abuddenb
Updated : January 20, 2024

The terminal menu's main way of installing software is through the Install-Application function.

Install-Application Install-BIOSUpdates Install-NewTeams Install-SMARTNotebookSoftware
Install-VeyonRoom Apply-DellCommandUpdates (in progress)

Install-Application Walkthrough:

  1. Uses the PS-Menu module, and the folders in the 'deploy/applications' directory to present a menu of available software.
  2. You can choose one or more applications using the up/down arrow keys to navigate, and [SPACE] to select. Press [ENTER] when finished.
  3. The script will copy the PSADT folder(s) to target computer(s) and then execute a scriptblock to install each application.

BIOS Updates

This function can be replaced by installing Dell Command Update as long as you configure the BIOS password to allow it to update BIOS.

Install-BIOSUpdates executes Dell BIOS .exe's silently, with a switch to prevent reboots.

Outputs flash log.

*See the menu overview's section on BIOS updates for more info.

Installing the New Teams client (work or school):

Install-NewTeams will run a script to purge Microsoft Teams Classic from the computer before installing the new Teams client and creating scheduled task to run on user login to provision the application.

The uninstall/purge Teams Classic script is: localscripts/Purge-TeamsClassic.ps1, but the Install-NewTeams function actually just runs the 'NewTeams' PSADT installation from the 'deploy/irregular' folder.

The script to install 'New Teams' is from github (source is included in the comment at top of the script file).

SMART Notebook:

Install-SMARTNotebookSoftware can install the SMART Learning Suite for either an instructor workstation in a classroom (with the SMART Ink Drivers for smartboard usage), or for an office (without SMART Ink Drivers).

Veyon:

Install-VeyonRoom will target a group of computers, and install Veyon client or master on them depending on parameter values.

After the installations, Install-VeyonRoom creates a Powershell script that you need to run while logged in to the master computer (either through RDP or in person) to add the student/client computers and make it possible to view/control them.

Applying updates through Dell Command | Update:

Apply-DellCommandUpdates attempts to use the /applyUpdates switch for Dell Command | Update to trigger installation of available updates for Dell Computers.
