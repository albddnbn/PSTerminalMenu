Getting device IDs for Autopilot:

At it's core, the Get-IntuneHardwareIDs function uses Get-WindowsAutopilotInfo.ps1 https://github.com/DvirEzra/Add_Devices_to_Windows_Autopilot.

It uses the ComputerName parameter to target groups of remote computers, and allows you to set the group tag automatically so you don't have to remember to do this in Autopilot. There's an example below of what the output looks like when you use the GroupTag parameter.

There is also an option to auto-import devices to Intune using Microsoft's Graph API.

Generating an .intunewin package (Intune Win32 App):

You can target a PSADT (Powershell App Deployment Toolkit) folder to create an .intunewin package, for software deployment through Intune.

Technically, it can be any folder, that has an installer or installation script / source files in it.

Generating Student Laptop Loan Documents:

The menu can generate copies of the Tech Loan Welcome Letter and Agreement Form.

It will ask you to input the student's name, 700 number, and the device's serial number.

New-TechLoanDocs will check the 'Techloan/laptops.csv' file to attempt to auto-match the serial number with an asset tag and device model.

If the script can't make a match - it will ask for input for asset tag and device model.
