Configuration File (config.json)

Author : abuddenb
Updated : January 14, 2024
The menu's configuration file is: 'supportfiles\config.json', and it allows you to change the menu structure, add functions, and change the values of some of the environment variables that are set.

Terminal menu configuration file:

categories: allows you to define the categories listed when running the menu, after it's loaded modules/etc.

*Don't forget the comma before the new category entry.

Each category has a list as it's value, and each element in the list appears as an option after selecting list's category.

Menu.ps1 replaces any - with spaces to make it look a bit nicer, but it needs to be able to use Get-Command and Get-Help to get info on a function, to work properly.

Adding functions to the menu:

Basic steps include:

  1. Create a .ps1 file in the 'functions' directory of the menu.
  2. Create a function, ideally with same name as the .ps1 file, inside of it.
  3. Create comment block at the beginning of the file, underneath the beginning of the function and above parameters, that gives a basic description of the function and any parameters therein.

    4. The text underneath each .SECTION in the comment block, must be indented. Parameter names must be one space after .PARAMETER on the same line.

    5. It's ok to put the function name into multiple categories, it will just appear after choosing any of those categories in the menu.
